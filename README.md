# ISM2411 Data Cleaning Copilot Project

## Project Overview
This project is a Python-based data cleaning pipeline designed to process messy sales data. It standardizes column names, removes invalid entries, handles missing values, and formats text columns for analysis.

## Repository Structure
- `data/raw/`: Contains the original messy dataset (`sales_data_raw.csv`).
- `data/processed/`: Contains the cleaned dataset generated by the script (`sales_data_clean.csv`).
- `src/`: Contains the Python source code (`data_cleaning.py`).
- `reflection.md`: A summary of how I used GitHub Copilot to assist with coding and debugging.

## How to Run
1. Ensure you have Python and pandas installed:
   ```bash
    pip install pandas
2. Run the cleaning script from the project root:
    python src/data_cleaning.py
3. The cleaned file will generate at:
    data/processed/sales_data_clean.csv
